## 마케팅 퍼널 구조

```
Threads 게시물 (훅 → 공감 → CTA)
        ↓
   Hub Page (허브페이지)
        ↓
  상세 페이지 (전자책 설명)
        ↓
     구매 (Google Form)
```

---

## 핵심 전략 4원칙

1. **책 속 기법 시연** — 스크립트 설득 기법을 Threads 문구 자체에 적용
2. **5주 시리즈 운영** — 5단계 세일즈 사이클 (라포→브리핑→갈망→거절→클로징) 주차별 전개
3. **5가지 콘텐츠 유형** — 공감형·장면형·반전형·원칙형·CTA형 로테이션
4. **원본 문장 재활용** — 책 머리말·에필로그·각 장 핵심 문장을 훅으로 직접 사용
