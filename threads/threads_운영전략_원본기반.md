**퍼널 구조**
```
Threads 게시물 → Hub Page → 상세 페이지 → 구매 (Google Form)

**핵심 설계 원리**
> 책이 독자에게 쓰는 설득 기법을 Threads 문구 자체에 그대로 시연한다.

## 🗂️ 콘텐츠 유형 5가지

책의 5단계 구조를 그대로 5가지 Threads 콘텐츠 유형으로 매핑한다.

| 콘텐츠 유형 | 대응 챕터 | 핵심 감정 | 게시 목적 |
|---|---|---|---|
| **A. 공감형** | 머리말 / 에필로그 | "나 얘기다" | 팔로우 유도 |
| **B. 장면형** | 2장 매물브리핑 | "이렇게 말하면 되는구나" | 전문성 증명 |
| **C. 반전형** | 4장 거절대응 / 부록 | "이런 시각이 있었네" | 저장·공유 유도 |
| **D. 원칙형** | 3장 갈망증폭 | "핵심을 꿰뚫는다" | 신뢰 축적 |
| **E. CTA형** | 5장 클로징 | "나도 갖고 싶다" | 허브페이지 클릭 |

## ✍️ 콘텐츠 유형별 템플릿

### A. 공감형 — "나 얘기다" 시리즈

**목적:** 분양 컨설턴트의 일상적 고통에 공감 → 팔로우·저장 유도

**구조:**
```
[훅] 고통의 장면을 구체적으로 묘사
[본문] 그 고통이 왜 생기는지 1~2문장
[반전] 준비만으로 달라진다는 사실
[CTA] 링크 또는 "저장해두세요"
```

**예시 1 (머리말 기반)**
```
처음 픽업 나가던 날,
어색한 침묵을 채우려 라디오 볼륨만 높였다.

현장에서 열심히 설명했는데
고객 눈빛은 점점 멀어지고.

돌아오는 차 안에서
"좀 더 생각해볼게요"를 들으며
집에 돌아와 혼자 되새기던 밤.

이건 능력의 차이가 아니다.
준비의 차이다.

👉 [링크]
```

**예시 2 (에필로그 기반)**
```
이 일 오래 하면 알게 된다.

계약서보다 먼저 쌓이는 게 신뢰라는 걸.
그 신뢰가 이 일을 오래 할 수 있게 하는
유일한 자산이라는 걸.

오늘도 누군가를 픽업하러 가는 길,
차 안에서 이 말이 문득 떠오른다면
그것으로 충분하다.

📌 저장해두고, 내일 현장 가기 전에 꺼내 보세요.
```

### B. 장면형 — "이렇게 말하면 된다" 시리즈

**목적:** 실제 현장에서 쓸 수 있는 말 한마디를 제시 → 저장·공유 유도

**구조:**
```
[상황] 현장에서 자주 마주치는 그 장면
[잘못된 말] 대부분이 하는 표현
[올바른 말] 책의 스크립트 or 원리
[이유] 왜 이 말이 통하는지 (심리기제 쉽게 설명)
[CTA] "이런 스크립트가 50개 있습니다"
```

**예시 1 (스크립트 #11 기반 — 미래 시각화)**
```
거실 통창 앞에서 이렇게 말하고 있지 않은가.

❌  "남향이라 채광 좋습니다."

고객의 눈빛이 움직이는 말은 따로 있다.

✅  "아침에 햇살 받으면서 커피 한 잔 마시면
    카페가 따로 없겠죠?"

스펙을 말하지 마라.
장면을 그려줘라.

사람은 숫자로 집을 사지 않는다.
그 집에서 살아가는 자신의 모습을
상상할 수 있을 때 갖고 싶다는 감정이 생긴다.

📎 이런 장면 만드는 스크립트 50개 → [링크]
```

**예시 2 (스크립트 #04 기반 — 니즈 파악)**
```
차 안에서 고객에게 이렇게 묻고 있는가.

❌  "어떤 집 찾고 계세요?"
    → 고객이 무슨 말 해야 할지 막막해한다.

✅  "지금 사시는 집에서
    탈출하고 싶은 점 딱 한 가지만 말씀해 주시겠어요?
    오늘 그 문제 해결하는 게 목표니까요."

'딱 한 가지만'이라는 제한이 핵심이다.
고객은 가장 절박한 문제를 먼저 꺼낸다.
그 한마디가 오늘 투어 전체의 나침반이 된다.

📎 픽업부터 클로징까지 스크립트 50개 → [링크]
```

**예시 3 (스크립트 #19 기반 — 이득의 구체화)**
```
옵션을 소개할 때 이렇게 말하고 있다면.

❌  "풀옵션이라 편리하게 잘 갖춰져 있어요."

이 한 마디로 바꿔라.

✅  "시스템 에어컨 3대 무상인데,
    이게 별거 아닌 것 같아도 600만 원 이상 아끼시는 거예요.
    다른 곳들은 옵션 별도가 많거든요."

막연한 혜택보다
구체적인 숫자가 10배 더 강하다.

'풀옵션'이라는 말은 추상적이지만
'600만 원을 아낀다'는 말은 즉각 와닿는다.

📎 현장에서 바로 쓰는 스크립트 50개 → [링크]
```

### C. 반전형 — "이런 시각도 있다" 시리즈

**목적:** 통념을 뒤집는 관점 제시 → 저장·공유·댓글 유도

**구조:**
```
[통념] 대부분이 당연하게 여기는 생각
[반전] 반대 관점 or 더 정확한 사실
[근거] 책의 심리기제 or 현장 경험
[결론] 실용적 적용점
[CTA] "이 원칙이 담긴 책"
```

**예시 1 (부록 기반 — 거절 해석)**
```
"좀 더 생각해볼게요"

이 말을 거절로 듣고 있다면
50%의 계약을 놓치고 있다.

이 말의 진짜 의미는 대부분 이것이다.

— 결정이 두려운 것
— 배우자 눈치가 보이는 것
— 지금 자리에서 결정하는 게 부담스러운 것

집이 마음에 없는 게 아니다.

거절을 반박하려 할 때 대화가 닫힌다.
거절 뒤에 숨은 진짜 말을 들을 때
계약이 열린다.

📎 거절 유형별 대응 스크립트 10가지 포함 → [링크]
```

**예시 2 (스크립트 #37 기반 — 완벽주의 해체)**
```
"완벽한 집을 찾겠다"는 고객에게
쓰는 말이 있다.

"저 이 일 오래 했는데,
100% 마음에 드는 집 계약한 분
한 명도 못 봤어요.

오늘 보신 것들 중에 80%가 맞는다면,
그게 고객님한테 맞는 집이에요.
남은 20%는 살면서 채우는 거고요."

완벽한 집을 기다리는 건
선택 회피의 합리화다.

80%면 충분하다는 기준이
결정을 내려도 된다는 심리적 허가가 된다.

📎 갈망증폭부터 클로징까지 스크립트 → [링크]
```

**예시 3 (스크립트 #06 기반 — 역설적 프레이밍)**
```
분양 컨설턴트가 현장 입구에서
이 말을 건네면 계약률이 올라간다.

"오늘 제 역할은 집을 파는 게 아니에요.
여러 곳 직접 보시고
고객님이 '이거다' 싶은 걸 찾을 수 있게
옆에서 도와드리는 겁니다.

마음에 안 드시면 솔직하게 말씀해 주세요."

압박이 사라지면
고객은 오히려 더 열린 마음으로 집을 본다.

좋은 집을 만나면
고객 스스로 원한다는 감정이 더 선명해진다.

역설이지만 사실이다.

📎 라포형성부터 클로징까지 50개 스크립트 → [링크]
```

### D. 원칙형 — "본질을 꿰뚫는다" 시리즈

**목적:** 전문적이고 본질적인 인사이트 제공 → 팔로우·신뢰 축적

**구조:**
```
[선언] 강력한 한 줄 원칙
[풀이] 3~4줄로 설명
[현장 적용] "이 원칙이 현장에서 이렇게 작동한다"
[CTA] 가볍게 연결
```

**예시 1**
```
차 안 30분이 계약을 결정한다.

고객은 첫 번째 현장에 도착하기 전에
이미 마음을 열거나 닫는다.

라포가 형성되지 않은 상태에서
아무리 좋은 매물을 보여줘도
고객의 눈에는 제대로 들어오지 않는다.

방어적인 상태로 집을 보는 사람은
단점만 찾는다.

픽업 직후 20~30분,
이 시간이 당신의 황금 타임이다.
```

**예시 2**
```
거절 대응에는 순서가 있다.

1. 인정하라
2. 파악하라
3. 전환하라

고객의 말을 먼저 부정하는 순간
대화는 닫힌다.

"그렇게 느끼시는 거 당연해요"
이 한 마디가 먼저다.

컨설턴트가 이겼다는 느낌이 들면
고객은 지지 않으려 더 강하게 버틴다.
```

**예시 3**
```
스크립트를 외우는 게 목표가 아니다.

고객의 심리를 이해하고
그에 맞게 말을 설계하는 능력을 갖추는 것.

외운 말은 어색하게 들린다.
이해한 말은 자신의 말로 나온다.

현장에서 어색하게 들리는 스크립트는
쓰지 않는 것이 낫다.

자신의 언어로 완전히 녹아들었을 때만
진정성이 전달된다.
```

### E. CTA형 — 구매 전환 게시물

**목적:** 허브페이지 클릭 → 구매 전환

**구조:**
```
[공감] 독자의 고통 요약
[가치 제시] 책이 주는 것
[증거] 구체적 숫자 or 구성
[행동 유도] 링크 + 구체적 지시
```

**예시 1**
```
하루 종일 같이 다녔는데
"좀 더 생각해볼게요"로 끝난 날.

그 공허함이 익숙하다면.

픽업 → 브리핑 → 갈망 증폭 → 거절 대응 → 클로징
5단계 72개 실전 스크립트 + 반박 시나리오 10개

신축빌라 분양 컨설턴트를 위해
현장 하루의 흐름 전체를 담았습니다.

오늘 바로 쓸 수 있습니다.

👇 상세 내용 확인
[링크]
```

**예시 2 (희소성 + 후회 회피)**
```
이 일 하면서 가장 많이 듣는 말.

"그때 살걸."

계약을 망설이는 고객 앞에서
무슨 말을 해야 할지 몰라
그냥 보내고 나서 혼자 되새기는 밤.

이 책에 담았습니다.

스크립트 #27 — 후회 회피 화법 포함
총 50개 스크립트 + 10가지 거절 반박 시나리오

👇 [링크]

## 📊 5주 시리즈 콘텐츠 플랜 (런칭 집중 운영)

책의 5단계 구조를 5주 시리즈로 전개한다.
매 주 말 CTA 게시물로 허브페이지 연결.

### 1주차: 라포 형성 — "차 안 30분의 기술"
- 월: 공감형 — 픽업 당일 어색한 침묵의 고통
- 화: 장면형 — 음료 한 잔이 만드는 상호성 (스크립트 #01)
- 수: 원칙형 — 차 안은 이동 수단이 아니라 계약의 무대
- 목: 반전형 — 고객이 말하게 만들어야 계약이 열린다
- 금: CTA형 — "1단계 라포 스크립트 10개 포함"

### 2주차: 매물 브리핑 — "스펙 말고 장면을 팔아라"
- 월: 공감형 — "남향입니다"만 말하다 고객 잃은 날
- 화: 장면형 — Feature→Benefit→Image 공식 (스크립트 #11)
- 수: 원칙형 — 차 안에서 들은 말을 현장에서 다시 꺼내는 이유
- 목: 반전형 — 단점을 인정하면 신뢰가 올라간다
- 금: CTA형 — "브리핑 스크립트 10개 포함"

### 3주차: 갈망 증폭 — "'좋아요'를 '지금 결정'으로 바꾸는 법"
- 월: 공감형 — 고객이 마음에 들어 하다 그냥 간 날
- 화: 장면형 — 고객의 발걸음이 느려지는 그 순간 (스크립트 #21)
- 수: 원칙형 — 긴박감은 밀어붙이는 것이 아니다
- 목: 반전형 — 압박을 풀 때 오히려 결정이 생긴다
- 금: CTA형 — "갈망 증폭 스크립트 10개 포함"

### 4주차: 거절 대응 — "'아니오' 뒤에 숨은 진짜 말"
- 월: 공감형 — "배우자랑 상의해봐야 해요"를 들은 날
- 화: 장면형 — 거절 대응 3단계 원칙 (부록 기반)
- 수: 원칙형 — 거절의 진짜 이유를 먼저 파악하라
- 목: 반전형 — 10가지 거절 뒤에 숨은 진짜 의미
- 금: CTA형 — "거절 반박 시나리오 10개 포함"

### 5주차: 클로징 — "도장 찍는 마지막 10분"
- 월: 공감형 — "하겠어요"와 서명 사이 고객 마음이 흔들리는 그 간격
- 화: 장면형 — "계약하실 건가요?" 대신 "언제가 편하세요?" (스크립트 #41)
- 수: 원칙형 — 클로징은 압박이 아니라 결정을 실행하도록 돕는 것
- 목: 반전형 — 계약 이후가 진짜 시작인 이유
- 금: CTA형 — "전체 50개 스크립트 구매 안내"

## 🪝 훅(Hook) 패턴 6가지 × 전자책 맞춤 예시

| 패턴 | 구조 | 예시 |
|---|---|---|
| **고통 직구** | "당신도 이런 적 있지 않은가" | "처음 픽업 나가던 날, 어색한 침묵을 채우려 라디오 볼륨만 높였다." |
| **숫자 충격** | 구체적 수치로 관심 유발 | "스크립트 50개. 심리기제 설명 50개. 거절 시나리오 10개." |
| **질문 던지기** | 독자가 스스로 답하게 만들기 | "고객이 '좀 더 생각해볼게요' 할 때, 당신은 뭐라고 하는가?" |
| **역설 제시** | 통념을 뒤집는 한 마디 | "집을 팔지 않겠다고 선언할 때 계약률이 올라간다." |
| **인용 활용** | 책 속 강력한 문장 그대로 | "집은 사는 것이 아니라 인연이 닿아 주인이 되는 것이다." |
| **before/after** | 이전 vs 이후 대비 | "❌ '남향입니다' → ✅ '아침에 햇살 받으면서 커피 한 잔 마시면 카페가 따로 없겠죠?'" |

## 🔗 허브페이지 CTA 문구 패턴

게시물 말미에 반드시 하나의 CTA를 붙인다. 아래 패턴을 돌아가며 사용.

```
📎 스크립트 전체 목차 확인 → [링크]
📌 픽업부터 클로징까지 50개 스크립트 → [링크]
👇 현장에서 바로 쓰는 스크립트 50개 → [링크]
🔗 상세 내용 + 발췌본 무료 다운로드 → [링크]
📖 전자책 미리보기 (발췌본 무료) → [링크]
```

> **원칙:** 링크는 게시물 본문 마지막에 딱 하나만. 여러 링크를 나열하면 클릭률이 떨어진다.

- [ ] 해시태그는 3개 이하인가?

- [ ] 저장하고 싶은 게시물인가?

## 🔑 핵심 자산: 책에서 바로 쓸 수 있는 훅 문장 목록

- `"통한 말을 기억해 두십시오."`
